How to Integrate the Translation Widget

Follow these simple steps to add translation capabilities to your website:

<!-- Add this script tag to your HTML head or before closing body tag --> <script src="/dist/widget/widget.iife.js" data-target-website="https://your-website.com" data-debug="false"> </script>
Note: The widget will automatically initialize when the page loads. For manual control, add data-auto-start="false" and use window.initTranslationWidget().

Our Features

Easy Integration

Simply add one script tag to your website and the translation widget is ready to use. No complex setup or configuration required.

Hover to Translate

Hover over any text element to see translation options. Click the translate button to open the translation editor.

Multiple Languages

Support for multiple languages including English, Spanish, French, German, and Italian. Easy to extend with more languages.

Import & Export

Export translations as JSON or CSV files for sharing with translation partners. Import translations from external sources.

Version Control

Track translation history with version control. See who made changes and when. Revert to previous versions if needed.

User Management

Role-based permissions for translators and administrators. Secure authentication with support for OAuth providers.

Getting Started

Ready to start translating your website? Here's what you need to do:

  1. Download the Widget: Get the latest widget.js file from the build output
  2. Add the Script: Include the script tag in your website's HTML
  3. Configure: Set your target website URL and other options
  4. Authenticate: Login with your translator credentials
  5. Start Translating: Hover over text and start adding translations

Technical Documentation

For developers integrating the widget, here are the available configuration options:

// Manual initialization with custom options window.initTranslationWidget({ targetWebsite: 'https://your-website.com', debug: false, containerId: 'custom-widget-container', autoStart: true }); // Access widget instance const widget = window.TranslationWidget; // Export current translations widget.exportTranslations().then(translations => { console.log('Current translations:', translations); }); // Destroy widget widget.destroy();
